
# Estudos ADS

Repositório voltado ao estudo de programação durante a graduação em Análise e Desenvolvimento de Sistemas (ADS). O foco principal é desenvolver lógica de programação e organização de código por meio de exercícios práticos.


## Objetivo

Construir uma base sólida em programação, evoluindo gradualmente de exercícios simples para soluções mais estruturadas e próximas de aplicações reais.


## Conteúdos

* Entrada e saída de dados
* Estruturas condicionais
* Operações matemáticas
* Lógica de programação
* Organização de código
* Versionamento com Git


## Projetos

### Análise Financeira

Programa desenvolvido em C++ que realiza:

* leitura de dados do usuário
* cálculo de renda e gastos
* cálculo de percentual de despesas
* classificação da situação financeira


## Tecnologias

* C++
* Visual Studio Code
* Git e GitHub

## Evolução

O repositório será atualizado conforme o avanço nos estudos, incluindo:

* estruturas de repetição
* funções
* organização modular de código
* projetos mais completos
